== Description ==

This plugin allows you to manage your upcoming website launch and site offline pages. It comes up with three templates, a countdown timer, and customizable CSS. All the templates are fully responsive in their design. Coming Soon provides you with the visionary and well-formatted upcoming pages.

= Main Features =

* Turns on, in just a single click.
* Fully Responsive in design with three templates.
* Html5 and CSS3 support.
* Works with any WordPress Theme.

== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't need to leave your web browser. To do an automatic install of Wordpress Coming Soon Page, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Wordpress Coming Soon Page" and click Search Plugins. Once you've found our Wordpress Coming Soon Page plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking "Install Now".


= Manual installation =

The manual installation method involves downloading our Wordpress Coming Soon Page and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).


= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
